# Quick install

```make install```

# Deploy it (example)

- change refresh.sh in your scripts folder (for example move it in ~/.scripts)

- add in your .xinitrc sh $HOME/.scripts/refresh.sh > /dev/null 2>&1 or use systemd 

# Dependence

- imagemagick: build wallpaper
- feh: display wallpaper into background
